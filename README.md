# Assignment-6 – Route Academy (🎮 Gaming Landing Page)

A modern gaming landing page built using HTML5, CSS3 & Bootstrap, delivering an immersive UI inspired by real eSports platforms.

## 📌 Features
* Fully responsive design (Mobile → Tablet → Desktop)
* Built using Bootstrap & its components
* Clean and structured layout
* Modern gaming-style UI with strong visual hierarchy
* Smooth animations and hover effects
* Infinite scrolling effect
* Visual enhancements using CSS filters (blur, overlays)

---

## 🧱 Sections
* Hero Section – Engaging intro with call-to-action
* Our Latest Games – Carousel-based game showcase
* What We Do – Services and features
* Our Team – Team members section
* Contact Us – Form with styled inputs
  
---

## 🛠️ Technologies Used
* HTML5
* CSS3
* Bootstrap (Navbar, Carousel, Forms, Buttons)
* Bootstrap Utilities & Components
* Google Fonts

---

## 🎯 What I Practiced

* Using Bootstrap components (Navbar, Carousel, Forms, Buttons)
* Customizing Bootstrap to match a gaming UI style
* Implementing responsive layouts quickly using Bootstrap grid
* Creating smooth UI interactions
* Applying visual effects like blur and overlays
* Structuring a multi-section landing page
  
## 📱 Responsive Breakpoints

* Mobile: Default
* Tablet: min-width: 768px
* Desktop: min-width: 1024px
* Large Screens: min-width: 1370px
